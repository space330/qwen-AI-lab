# Qwen Agent Lab 1.2.0

发布日期：2026-06-04

## 新增

- 多轮对话上下文：摘要加最近 8 个完整问答轮次。
- 会话管理：新建、搜索、切换、自动标题、重命名和删除。
- 按模式隔离会话，每个会话保存独立模型、消息和一个附件。
- 本地个人档案：昵称、内置头像、默认模型和默认模式。
- 本地数据导出与清除。
- 长会话后台自动摘要和失败降级。

## 存储升级

- 会话数据从 `localStorage v6` 迁移到原生 IndexedDB。
- 迁移成功前保留旧数据。
- 布局宽度等轻量 UI 设置继续保存在 localStorage。
- IndexedDB 不可用时进入临时内存模式，并在个人档案中提示。

## 接口升级

- `/api/qwen/chat` 与 `/api/qwen/chat/stream` 支持 `context`。
- 新增 `POST /api/qwen/conversation-summary`。
- 后端限制历史角色、轮次、单条长度和总上下文大小。
