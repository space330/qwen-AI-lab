## 二、V2.0 「真·Agent」落实情况

将 Agent 模式从占位升级为**真正的 ReAct 工具调用循环**。

### 实现内容
- **执行核心**：`server/agentExecutor.js` 的 `runAgent`（async generator），与传输/服务商解耦，便于单测。
- **路由分流**：`server/index.js` 的 `/api/qwen/chat` 与 `/api/qwen/chat/stream` 均按 `mode` 区分；Agent 走 `executeAgent`（流式/非流式共用同一逻辑），Chat 走单次模型调用。
- **内置工具**：`calculator`（自实现解析器，非 eval，除零报错）、`file_reader`（按行读取会话附件）、`web_search`（默认 `provider=qwen`，复用 DashScope key 走千问内置 `enable_search`，零新增凭据；可选 `bocha`/`mock`）。
- **统一 toolSteps 契约**：`step/tool/status/args/result/error/durationMs`，贯通流式事件、最终响应、前端状态与 IndexedDB 持久化。
- **循环限制**：`AGENT_MAX_ITERATIONS`（默认 5）、`AGENT_MAX_TOOL_STEPS`（默认 8），超限强制收束并标记 `agentTruncatedReason`（`iteration_limit` / `tool_step_limit`）。
- **错误恢复**：工具抛错 / 未知工具 → `tool_error` 回填，不崩溃；步骤参数/结果/错误经 HTML 转义渲染。

### 验收结果（源 `docs/v2.0-acceptance.md`）
- 16 项验收（单工具、多步连续、多工具同轮、结果回填、错误恢复、未知工具、步骤超限、Chat 不调工具、流式/非流式接口、toolSteps 契约与持久化、真实联网搜索、搜索失败不崩、Mock 标记、版本统一）——**全部通过**。
- 问题清单 `docs/v2.0-issues.md`：P0-1/P0-2、P1-3/4/5、P2-6/7/8 **全部修复并通过**。

## 三、V2.1 「输出与生态增强」落实情况

### 实现内容
- **任意范围多选导出**：AI 面板「下载」图标进入 Export Mode，消息左侧复选框、底部浮动 Dock（全选 / HTML 下载 / PDF 打印 / 取消）。
- **主题化 Standalone HTML 导出**：`src/utils/exportReport.js` 把选中卡片包成自包含 HTML，内嵌完整 Lysandra 暗色玻璃主题与图表样式，离线可独立渲染。
- **PDF 打印**：`@media print` 隐藏顶栏/侧栏/输入坞/复选框/Dock，仅渲染选中卡片，`break-inside: avoid` 防切割，正文转高对比黑字。
- **五大图表自愈**：`src/components/chartPolisher.js`——折线单点/非数字退避表格、柱状裁剪 `[0,100]%`、散点裁剪 `[2,98]%`、饼图剔除非正切片、除零/NaN 守卫。
- **实时「吐字式」流式**：后端 `streamQwenOnce` 经 `delta` 事件流式回传 token，前端剥标签打字机动画 + 光标，结束换入渲染后 HTML。
- **防崩环境变量**：`server/config.js` 的 `parseIntEnv`/`parseFloatEnv`，非法值安全回退默认。
- **高精度 CSV 分词**：`src/utils/fileParser.js` 基于状态机的 `tokenizeCsv`，正确处理单元格内换行与带逗号 quoted 字段。

### 验收结果（源 `docs/v2.1-acceptance.md`）
- 9 项验收（多选开启、勾选计数同步、全选/取消、主题化 HTML 导出、PDF 打印裁剪、图表自愈、流式吐字、防崩环境变量、复杂 CSV 解析）——**全部通过**。
- 安全审计：导出标题 `escapeHtml` 防 XSS、卡片内容取自白名单净化节点、图表除零/NaN 守卫——均已覆盖。
